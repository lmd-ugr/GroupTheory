# Teoría-de-Grupos
Repositorio para TFG

Implementación de librería de python sobre teoría de Grupos de la asignatura ÁlgebraII [Grado en Matemáticas](http://grados.ugr.es/matematicas/) de la [Universidad de Granada](http://www.ugr.es).

Extensión de la librería de [pedritomelenas](https://github.com/pedritomelenas), creada por 
[Naftali Harris](http://www.naftaliharris.com) y disponible en 
[absalg](https://github.com/naftaliharris/Abstract-Algebra)
